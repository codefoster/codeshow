codeShow()

codeShow is a Windows 8 HTML/JS project with the express goal of demonstrating simple development concepts for the Windows 8 platform.

This is not necessarily a project to learn about end-to-end app architecture, but when you're looking for how to do things like access the camera, implement an asymmetric ListView, or handle errors look to codeShow.

The complete source code to the project is available at http://codeshow.codeplex.com.
